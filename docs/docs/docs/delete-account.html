Suppression de compte — BIZZ DASHBOARD

Dernière mise à jour : 27/02/2026

Option 1 — Depuis l’app (automatique)

Ouvrez l’app → Paramètres → Supprimer mon compte.

Option 2 — Si vous n’avez plus accès à l’app

Envoyez un email à support@TONDOMAINE.COM avec l’objet Suppression de compte BIZZ DASHBOARD.

Ce qui est supprimé

  • Compte (authentification)
  • Données cloud associées (clients, revenus, dépenses)
